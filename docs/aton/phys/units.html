aton.phys.units

Description

This module contains useful constants and conversion factors.

Constant values come from the 2022 CODATA Recommended Values of the Fundamental Physical Constants.

Greek letters are romanized.

 42## Conversion factors

 44### Energy
 45Note that `cm1` refers to cm$^{-1}$.
 46"""
 47
 48import numpy as np
 49
 50eV_to_meV   = 1000.0
 51meV_to_eV   = 0.001
 52meV_to_cm1  = 8.0655
 53cm1_to_meV  = 1.0 / meV_to_cm1
 54eV_to_J     = 1.602176634e-19
 55J_to_eV     = 1.0 / eV_to_J
 56meV_to_J    = meV_to_eV * eV_to_J
 57J_to_meV    = J_to_eV * eV_to_meV
 58Ry_to_eV    = 13.605693122990
 59eV_to_Ry    = 1.0 / Ry_to_eV
 60Ry_to_J     = 2.1798723611030e-18
 61J_to_Ry     = 1.0 / Ry_to_J
 62cal_to_J    = 4.184
 63J_to_cal    = 1 / cal_to_J
 64kcal_to_J   = cal_to_J * 1000.0
 65J_to_kcal   = 1 / kcal_to_J
 66
 67"""---
 68### Distance
 69Note that `A` refers to Angstroms.
 70"""

 74m_to_bohr   = 1.0 / bohr_to_m
 75A_to_bohr   = A_to_m * m_to_bohr
 76bohr_to_A   = 1.0 / A_to_bohr
